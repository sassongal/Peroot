# PROMPT: PEROOT BRAND ASSETS

## 1. Logo Generation Prompt

**Role:** Master Brand Designer
**Objective:** Create a high-fidelity visual identity for "Peroot".

**The Prompt (Copy & Paste):**

> "A professional, minimalist logo featuring the Hebrew word 'פֵּרוּט' clearly centered. The typography must include the nikud (vocalization dots): 'פֵּרוּט'. The font should be modern, sleek, and architectural. The aesthetic is 'Obsidian Silver': deep black textured glass background with polished metallic chrome lettering. 8k, cinematic lighting, sharp focus, luxury tech feel. Isolated on black."

---

## 2. Dynamic Icon System Command

**Role:** UI/UX Iconography Specialist
**Objective:** Generate a consistent set of minimal icons.

**The Command:**

> "Generate a set of 6 sleek UI icons (Magic Wand, History Clock, Paper Plane, Gear, Clipboard, Brain). Style: Sharp silver-chrome outlines on rounded obsidian glass squares. Minimalist, premium stroke weight, identical lighting across all icons. High resolution, transparent or black background."

---

## 3. Implementation Instructions

To use these in your site:

1. Place the generated images in `/public/assets/branding`.
2. Use `<img src="/assets/branding/logo.png" />` in your `Header` component.
3. Update the `favicon.ico` with the simplified icon version.
